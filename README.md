Project title
Business Card App

Description
Android app built with Jetpack Compose. The app shows a digital business card with name, role, phone number, email, and profile image.

Features
• Simple UI using Compose
• Profile image support
• Contact information display

How to run

Open project in Android Studio

Sync Gradle

Run the app on emulator or phone
